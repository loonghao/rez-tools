# rez-tools
A suite tool command line for rez.

# Installing
```cmd
pip install rez-tools
```

# This tool requires you to install rez in advance


---------------------------------------------
**maya.rt**
```yaml
command: maya
requires:
   - maya-2020
   - maya_usd
   - maya_mtoa
```
run command line
```cmd
rt maya
```
